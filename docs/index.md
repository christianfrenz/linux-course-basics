# Learn Linux — Ubuntu Docker Lab

A hands-on, step-by-step Linux tutorial running inside Docker.
Break things, reset in seconds, and learn by doing.

---

## What You'll Learn

21 lessons covering everything from your first terminal command to Docker and Git — all practiced inside a safe, disposable Ubuntu container.

| # | Lesson | Topics |
| --- | --- | --- |
| 01 | [Getting Started](lessons/01-getting-started.md) | First commands, getting help, the shell |
| 02 | [Navigating the Filesystem](lessons/02-filesystem-navigation.md) | Paths, cd, ls, pwd, tree |
| 03 | [Files & Directories](lessons/03-files-and-directories.md) | touch, mkdir, cp, mv, rm, find |
| 04 | [Viewing & Editing Files](lessons/04-viewing-editing-files.md) | cat, less, head, tail, grep, nano, vim |
| 05 | [Permissions & Ownership](lessons/05-permissions-ownership.md) | chmod, chown, chgrp, umask |
| 06 | [Process Management](lessons/06-process-management.md) | ps, top, htop, kill, jobs, bg, fg |
| 07 | [Package Management](lessons/07-package-management.md) | apt update, install, remove, search |
| 08 | [Networking Basics](lessons/08-networking-basics.md) | ip, ping, curl, wget, ss, DNS |
| 09 | [Shell Scripting](lessons/09-shell-scripting.md) | Variables, conditionals, loops, functions |
| 10 | [System Administration](lessons/10-system-administration.md) | Users, services, cron, logs, disk |
| 11 | [Troubleshooting](lessons/11-troubleshooting.md) | Common errors, diagnostics, fixes |

| 14 | [Regular Expressions](lessons/14-regular-expressions.md) | Patterns, character classes, quantifiers, grep -E |
| 15 | [Shell Customization](lessons/15-shell-customization.md) | .bashrc, aliases, PS1 prompt, PATH, functions |
| 16 | [SSH & Remote Access](lessons/16-ssh-remote-access.md) | Keys, config, scp, rsync, tunnels |
| 17 | [Git Basics](lessons/17-git-basics.md) | init, add, commit, branch, merge, remotes |
| 18 | [Docker Basics](lessons/18-docker-basics.md) | Images, containers, Dockerfile, volumes, compose |
| 19 | [Login Auditing](lessons/19-login-auditing.md) | last, lastb, lastlog, who, w, auth.log, journalctl |

Each lesson contains **concepts**, **commands with explained flags**, **hands-on exercises**, and a **challenge**.
